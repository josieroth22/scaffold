# Scaffold

Every family deserves a college strategy. Not just the ones who can afford a $10,000 counselor.

Live at [scaffoldcollegestrategy.com](https://scaffoldcollegestrategy.com).

## What is this?

Scaffold generates personalized, 20+ page college strategy documents for families. $50, one time. A parent fills out a 7-step intake form, we run their details through Claude (Fable 5) with verified school data injected, and they get a full strategy doc at a private permanent URL: school list with financial analysis, developmental roadmap, application materials plan, and a code-generated Monte Carlo simulation modeling 10,000 admission and cost outcomes. The finished plan is also emailed to the family.

## How it's built

Vanilla HTML/CSS/JS, Vercel serverless functions (Node.js), Upstash Redis, Anthropic API. The generation pipeline is: Tier 1 strategy brief → programmatic validation + 15-check AI quality review (with targeted fix and full-regeneration fallbacks) → Monte Carlo simulation → cost reconciliation → Tier 2 reference sections → final review. School data comes from Common Data Set filings (136 schools), the College Scorecard (1,492 schools), and hand-verified reference data (scholarships, honors programs, state aid for all 50 states + DC).

## Project structure

```
src/            Pages (index, intake, plan, admin) and api/ serverless functions
src/lib/        School data injection, programmatic plan validator, model config
data/schools/   1,492 school JSON files (CDS + Scorecard + reference data)
data/           State aid programs, financial aid facts, session status log
scripts/        Data parsing/fetching, pipeline test driver, test profiles
docs/           Project plan (roadmap, unit economics), test profiles
prompts/        Legacy v3 prompt template (pre-pipeline era, kept for history)
samples/        Legacy hand-run sample outputs (pre-pipeline era)
```
